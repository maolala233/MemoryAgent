ADR-003: 自适应向量索引
===========================

状态
----

已采纳 (Accepted)

日期
----

2024-10

上下文
------

少量记忆（< 100）时暴力搜索更快，大规模时 FAISS 索引更快。需要平滑切换。

决策
----

引入 ``AdaptiveVectorIndex``，根据数据量自动切换：
- < 100：numpy 暴力搜索
- >= 100：FAISS HNSW 索引

阈值由 ``promote_threshold`` 控制（默认 100）。

后果
----

- 小数据集用户无需配置，零成本最优
- 大规模数据自动获得 FAISS 加速
- 切换时有一次性的索引构建开销（~10ms/千条）
