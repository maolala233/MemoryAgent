Mandol 是什么？
===============

一句话版本
----------

Mandol 是一个为 AI 智能体提供记忆能力的系统。将文本数据添加到系统后，系统自动提取结构化信息并构建索引，之后可通过自然语言查询检索相关记忆。

详细说明
--------

Mandol 提供以下核心能力：

- 自动记录对话、文档等文本数据，并提取人物、地点、事件、知识点
- 基于语义相似度匹配同义表述（如"在家办公"与"远程办公"）
- 跨越多段会话检索相关信息
- 按不同视角检索（知识视角、事件视角等）

它能做什么？
------------

- 存储对话、文档等文本数据
- 自动提取人物、地点、事件、知识点
- 匹配语义相近的表述
- 跨越多段会话检索相关信息
- 按不同视角检索（知识视角、事件视角等）

它不能做什么？
--------------

- 不是 ChatBot，本身不会与用户对话
- 不生成新内容，只检索和整理已有的记忆
- 不替代数据库，是记忆层而非存储层
