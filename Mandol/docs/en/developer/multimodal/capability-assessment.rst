Multimodal Capability Assessment
===================================

.. note::

   This document was converted from ``multimodal-capability-assessment.md`` and describes Mandol's current capabilities and limitations in multimodal scenarios.

Key Findings
-------------

- Text processing capability: Complete
- Image processing: Supports image_path field, vectorized through vision Embedding models
- Multimodal mixing: Text and images can coexist in the same MemoryUnit
- Known limitations: UnifiedFactPipeline has unimplemented method call issues

For details, see the source file: ``multimodal-capability-assessment.md``.
